This folder contains analysis files that can be used to reproduce the plots that were generated for the manuscipt. The ElasticAnalysis.m file can be run to extract the data from the simulations to generate the plots, an the Figure.m file can be run with varying parameters to generate the different figures observed in the manuscript. The user should replace directory line on line 3 of the ElasticAnalysis.m script to refer to the directory where the output files generated from SimulationTaskList001.m are saved. Here, the output files from the simulations are contained in the folder 'Mesoscale_output'.
